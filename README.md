
# Society Pet Rights

Simple mobile-friendly pet rights assistant for Mangalam Anada.

## Run Locally

1. Install Python 3
2. Open terminal in this folder
3. Run:

pip install -r requirements.txt

4. Start app:

streamlit run app.py

5. Open browser:
http://localhost:8501

## Features
- Mobile-friendly
- Quick pet rights questions
- Simple chat-style answers
- Installable as web app shortcut
